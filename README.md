# quanttide-strategy

量潮战略管理

## 简介

quanttide-strategy 是量潮战略管理领域，涵盖战略规划、战略解码、战略执行与战略评估等全链路管理实践。

## 目录结构

```
domains/quanttide-strategy/
├── apps/               # 面向用户的可部署应用
├── packages/toolkit    # 领域共享库/工具集
├── examples/default    # 实验室——实验性/原型项目
│   └── apps/           # 实验室内的应用原型
├── docs/               # 领域文档
├── .agents/            # Agent skills 配置
├── LICENSE             # CC BY 4.0
├── README.md           # 本文件
├── AGENTS.md           # AI 工作指南
└── CHANGELOG.md        # 版本变更记录
```
